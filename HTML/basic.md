
# HTML基础知识

## 文档结构
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

## 常用标签
- 文本标签: `<h1>-<h6>`, `<p>`, `<span>`
- 列表标签: `<ul>`, `<ol>`, `<li>`
- 表格标签: `<table>`, `<tr>`, `<td>`
- 表单标签: `<form>`, `<input>`, `<button>`
