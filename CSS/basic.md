
# CSS基础知识

## 选择器
```css
/* 元素选择器 */
div { color: red; }

/* 类选择器 */
.class-name { font-size: 16px; }

/* ID选择器 */
#id-name { background: #fff; }
```

## 盒模型
```css
.box {
    width: 100px;
    height: 100px;
    padding: 10px;
    border: 1px solid #000;
    margin: 20px;
}
```
